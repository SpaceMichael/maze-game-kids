# Funny Maze Escape / 搞笑迷宮大逃走

這是一個給 `3-6` 歲小朋友玩的迷宮網站遊戲。

玩家可以選擇幫 `小老鼠` 或 `小熊` 逃出迷宮。路上有搞笑的 `噴水陷阱`，一踩中就會被噴濕，然後回到起點重新挑戰。

## 功能

- 小老鼠 / 小熊角色選擇
- 大按鈕和方向鍵控制
- 三個固定迷宮關卡
- 噴水陷阱、起點與出口
- 適合幼兒的可愛視覺風格
- 可執行的單元測試

## 操作方式

1. 直接打開 `index.html`
2. 選擇角色
3. 使用畫面箭咀按鈕或鍵盤方向鍵移動
4. 踩到 `💦` 會回到起點
5. 到達 `🧀` 或 `🍯` 就可以過關

## 測試

使用 Node 內建測試：

```powershell
node --test .\gameLogic.test.mjs
```
